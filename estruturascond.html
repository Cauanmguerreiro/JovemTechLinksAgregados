Estudo das Estruturas Condicionais em Python

As estruturas condicionais em Python permitem que o programa execute diferentes blocos de código dependendo de condições específicas. Neste módulo, você aprenderá a usar as estruturas **`if`**, **`elif`** e **`else`** para tomar decisões no seu código.

Passo 1: O que são Estruturas Condicionais?

As **estruturas condicionais** verificam uma condição e, dependendo do resultado (verdadeiro ou falso), executam um bloco de código ou outro. As principais estruturas condicionais são **`if`**, **`elif`** e **`else`**.

Passo 2: Estrutura básica do `if`

A estrutura **`if`** é a mais simples das estruturas condicionais. Ela verifica uma condição e executa o código se a condição for **verdadeira**.

Exemplo:

idade = 20
        if idade >= 18:
            print("Você é maior de idade.")

Neste exemplo, o código dentro do **`if`** só será executado se a variável idade for maior ou igual a 18.

Passo 3: Adicionando a estrutura `else`

A estrutura **`else`** é usada para especificar o que fazer **quando a condição do `if` for falsa**. Ou seja, se a condição do **`if`** não for atendida, o código dentro do **`else`** será executado.

Exemplo:

idade = 16
        if idade >= 18:
            print("Você é maior de idade.")
        else:
            print("Você é menor de idade.")

Aqui, se a idade for menor que 18, o código dentro do **`else`** será executado, mostrando "Você é menor de idade".

Passo 4: Usando o `elif` para múltiplas condições

O **`elif`** (else if) permite que você verifique várias condições em sequência. Se a condição do **`if`** for falsa, o Python verificará as condições do **`elif`**. Se alguma dessas condições for verdadeira, o código correspondente será executado. Caso contrário, o **`else`** será executado.

Exemplo:

idade = 25
        if idade < 18:
            print("Você é menor de idade.")
        elif idade < 60:
            print("Você é adulto.")
        else:
            print("Você é idoso.")

Neste exemplo, a condição **`if`** é verificada primeiro. Se for falsa, o **`elif`** é checado. Se todas as condições forem falsas, o código dentro do **`else`** é executado.

Passo 5: Combinando múltiplas condições com `and` e `or`

Você pode combinar várias condições usando os operadores lógicos **`and`** (e) e **`or`** (ou).

Exemplo com `and`:

idade = 20
        tem_cnh = True
        if idade >= 18 and tem_cnh:
            print("Você pode dirigir.")

A condição dentro do **`if`** só será verdadeira se **ambas as condições** forem verdadeiras.

Exemplo com `or`:

idade = 16
        tem_permicao = True
        if idade >= 18 or tem_permicao:
            print("Você pode entrar na festa.")

A condição será verdadeira se **uma das condições** for verdadeira.

  • Compreender e utilizar as estruturas condicionais if, elif e else para tomar decisões em seu código.
  • Utilizar operadores lógicos como and e or para combinar múltiplas condições.
  • Organizar o fluxo do seu programa de forma eficiente, utilizando condições aninhadas e múltiplas verificações.

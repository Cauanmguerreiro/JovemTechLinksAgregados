O que é o Python?

O Python é uma linguagem de programação poderosa e fácil de aprender, usada em diversas áreas como desenvolvimento web, automação, ciência de dados, inteligência artificial e muito mais.

Este módulo irá te guiar através da instalação do Python e do seu ambiente de desenvolvimento, além de te ensinar os fundamentos dessa linguagem.

Conteúdo do Módulo:

  • Instalando o Python
  • Configurando o ambiente de desenvolvimento
  • Primeiros passos com a sintaxe do Python
  • Escrevendo seu primeiro código Python

Objetivos deste Módulo:

Ao final deste módulo, você será capaz de instalar o Python e escrever seu primeiro código Python básico. Além disso, entenderá como configurar um ambiente de desenvolvimento para codificar eficientemente.

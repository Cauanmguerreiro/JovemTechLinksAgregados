Trabalhando com Variáveis

As variáveis em Python são utilizadas para armazenar valores que podem ser manipulados em diferentes partes do código. Neste módulo, você aprenderá a criar variáveis e a trabalhar com os tipos de dados primitivos do Python.

Conteúdo do Módulo:

  • O que são variáveis e como usá-las

    • Em Python, as variáveis são espaços na memória onde podemos armazenar valores que podem ser manipulados e usados em diferentes partes do código.

    Exemplo:

    nome = "Cauan"
        idade = 20

    Neste exemplo, a variável nome armazena um texto (string), e a variável idade armazena um número inteiro.

  • Tipos de dados em Python: string, inteiro, float, etc.

    • Python oferece diversos tipos de dados para armazenar diferentes informações, como string (texto), inteiro (números sem casas decimais), float (números com casas decimais), e booleano (True ou False).

    Exemplo:

    nome = "Cauan"   # string
        idade = 20         # inteiro
        altura = 1.75      # float
        is_estudante = True  # booleano
  • Operadores aritméticos e lógicos

    • Operadores aritméticos permitem realizar operações matemáticas, como soma (+), subtração (-), multiplicação (*), entre outros. Já os operadores lógicos comparam valores e retornam True ou False, como ==, !=, and e or.

    Exemplo de Operadores Aritméticos:

    a = 10
        b = 5
        soma = a + b  # Resultado: 15
        multiplicacao = a * b  # Resultado: 50

    Exemplo de Operadores Lógicos:

    x = 10
        y = 5
        resultado = x > y  # Resultado: True
  • Como converter tipos de dados

    • Você pode converter um tipo de dado em outro usando funções como int(), float() e str(). Por exemplo, converter uma string para número ou vice-versa.
